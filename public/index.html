Laobai API

老白API 一键安装中心

按系统复制命令即可安装。已支持 Linux/macOS 和 Windows（PowerShell）。脚本与版本化入口都在本域名下托管。

OpenCode

安装 OpenCode CLI，并可写入 opencode.json 的 API Key。

curl -fsSL https://dl.laobaiapi.cc/opencode-setup.sh | bash
irm https://dl.laobaiapi.cc/opencode-setup.ps1 | iex
脚本直链：/opencode-setup.sh 脚本直链：/opencode-setup.ps1

OpenClaw

支持全新安装或注入 laobai provider，并补齐 gateway token。

curl -fsSL https://dl.laobaiapi.cc/openclaw-setup.sh | bash
irm https://dl.laobaiapi.cc/openclaw-setup.ps1 | iex
脚本直链：/openclaw-setup.sh 脚本直链：/openclaw-setup.ps1

Codex

一键注入 laobai provider 到 Codex 配置，适配 Codex CLI / VSCode(Cursor) Codex 插件 / Codex App。

curl -fsSL https://dl.laobaiapi.cc/codex-setup.sh | bash
powershell -NoProfile -ExecutionPolicy Bypass -Command "iwr https://dl.laobaiapi.cc/codex-setup.bat -OutFile $env:TEMP\codex-setup.bat; cmd /c $env:TEMP\codex-setup.bat"
脚本直链：/codex-setup.sh 脚本直链：/codex-setup.bat

安装后验证

opencode --version
openclaw --version
codex --version

看到版本号即代表命令可用。

如果命令不存在，重开终端后再试一次，或检查 Node.js/npm 是否安装成功。

常见问题

Windows 提示脚本执行被禁止

先在 PowerShell 使用管理员身份执行：Set-ExecutionPolicy RemoteSigned -Scope CurrentUser，然后重试。

安装后提示找不到命令

一般是 PATH 未刷新。重开终端，或执行 refreshenv（如果装了 Chocolatey）。

脚本会不会更新？

会持续更新。建议使用页面提供的主入口链接，始终获取最新修复版本。
