OpenClaw 一键安装 / 注入配置（Rocky Linux）

目标：

  • 方式一：全新安装 OpenClaw，并生成一个可启动的单 provider（laobai）配置。
  • 方式二：对已安装 OpenClaw 注入 laobai provider，并把默认模型切到 laobai/gpt-5.3-codex。
  • 两种方式都会补齐最低启动要件：gateway.mode=local、gateway.auth.token、默认模型。

一行下载运行（推荐）

curl -fsSL https://dl.laobaiapi.cc/openclaw-setup.sh | bash

版本固定（v1）

curl -fsSL https://dl.laobaiapi.cc/v1/openclaw-setup.sh | bash

可选参数

bash openclaw-setup.sh --fresh
bash openclaw-setup.sh --inject
bash openclaw-setup.sh --help
